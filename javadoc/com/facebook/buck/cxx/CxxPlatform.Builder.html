Class CxxPlatform.Builder

  • java.lang.Object
    • com.facebook.buck.cxx.CxxPlatform.Builder
  • Enclosing class:
    CxxPlatform

    @NotThreadSafe
public static final class CxxPlatform.Builder
extends Object
    Builds instances of CxxPlatform. Initialize attributes and then invoke build() method to create immutable instance.

    Builder is not thread safe and generally should not be stored in field or collection, but used immediately to create instances.
